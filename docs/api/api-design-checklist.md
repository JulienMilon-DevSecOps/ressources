# 📡 API Design Checklist

## 🎯 Objectif
Vérifier la qualité d’une API avant publication.

---

## ✅ Checklist

### 🔌 Structure
- [ ] RESTful
- [ ] Endpoints cohérents

### 📦 Données
- [ ] JSON standardisé
- [ ] Pagination

### 🔐 Sécurité
- [ ] Authentification
- [ ] Autorisation

### ⚠️ Erreurs
- [ ] Codes HTTP corrects
- [ ] Messages explicites

### 📚 Documentation
- [ ] Swagger / OpenAPI
